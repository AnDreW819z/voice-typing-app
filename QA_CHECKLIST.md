# QA Checklist (v1.0.0 RC1)

Интеграционные проверки и тестирование пограничных случаев (Edge Cases) для ручного тестирования перед релизом.

| Сценарий | Шаги | Ожидаемый результат | Статус |
| :--- | :--- | :--- | :--- |
| **Базовый запуск** | Запустить `main.py` или `.exe`. Дождаться полной инициализации | Лог `[✔] Audio device and Keyboard hooks ready`. Модель загрузилась. Приложение слушает F8. | ⬜ |
| **Короткая диктовка** | Нажать F8, сказать «Привет», отпустить F8 | Текст напечатался в активное окно. Формат лога с телеметрией присутствует, `Total Time` в адекватных пределах | ⬜ |
| **Длинная диктовка** | Держать F8, диктовать 30+ секунд непрерывно, отпустить F8 | Весь текст корректно и без потери слов вставился. Метрики (Total, RTF) залогированы. Приложение не упало (нет OOM). | ⬜ |
| **Целевое: Блокнот/VS Code** | Открыть редактор, продиктовать русский текст | Кириллица вставлена корректно. Добавлен пробел в конце. | ⬜ |
| **Целевое: Telegram** | Открыть чат Telegram, сменить раскладку на английскую, продиктовать русский текст | Текст вставился по-русски, без "кракозябр". Telegram не перехватил `F8` (благодаря `suppress_hotkey`). | ⬜ |
| **Стресс: Спам клавиши** | Очень быстро нажимать и отпускать F8 десятки раз в течение 3 секунд | Записи короче `min_recording_duration` отбраковываются (RecordingValidationError). Очередь не забивается. Утечек ресурсов нет. | ⬜ |
| **Отказ: Нет микрофона** | Отключить все микрофоны в ОС, нажать F8 | Лог ошибки инициализации звука. `AppState` сбрасывается в `IDLE`. Приложение **не** закрывается (ждет подключения микрофона). | ⬜ |
| **Отказ: Фокус окна (UAC)** | Открыть Диспетчер задач (или другое окно с правами Администратора), нажать F8 и диктовать | Текст не вставляется. В логе `Failed: target window rejected input`. Приложение **не** падает, переходит в `IDLE`. | ⬜ |
| **Отказ: Тишина** | Зажать F8, помолчать 3 секунды, отпустить F8 | Запись бракуется (уровень RMS ниже `silence_threshold`). Инференс Whisper не запускается (экономия GPU). Текст не выводится. | ⬜ |
| **Graceful Degradation (CPU)** | Удалить CUDA-библиотеки или запустить на ПК без GPU | Лог `Fallback на CPU (int8)...`. Модель успешно грузится на процессоре. Распознавание работает (хоть и медленнее). | ⬜ |
| **Проверка PyInstaller** | Собрать `.exe` через `build.spec` и запустить на чистой ОС Windows 11 | Приложение запускается. Встроенная модель `ctranslate2/faster-whisper` находит свои токенизаторы. Все работает без установленного Python. | ⬜ |
