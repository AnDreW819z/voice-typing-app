# Voice Typing App (v1.0.0)

Production-ready, полностью офлайн-приложение для Windows 11, преобразующее речь в текст "на лету" с использованием аппаратного ускорения NVIDIA (CUDA) и библиотеки `faster-whisper`. 

## Особенности
- **Push-To-Talk**: Запись работает только при зажатой горячей клавише (по умолчанию `F8`).
- **Сверхбыстрый инференс**: Использует модель `large-v3-turbo` и `compute_type="float16"` для мгновенного распознавания речи.
- **Интеграция в ОС**: Автоматический ввод распознанного текста в любое активное окно с полной поддержкой кириллицы независимо от текущей раскладки клавиатуры.
- **Отказоустойчивость**: Встроенный Fallback на CPU (при сбоях CUDA), игнорирование случайных нажатий, защита от залипаний и изоляция сессий.
- **Локальность**: 100% приватность. Звук обрабатывается в оперативной памяти.

## Системные требования
- **ОС:** Windows 10 / Windows 11 (64-bit)
- **GPU (Опционально, но рекомендуется):** NVIDIA GPU с поддержкой CUDA 11.x/12.x и объемом VRAM не менее 4-6 ГБ для комфортной работы модели `large-v3-turbo`.
- **CPU (Fallback):** Любой современный процессор (распознавание происходит медленнее через `int8`).
- **Python:** Строго версии **3.10 – 3.12** (рекомендуется **3.11**). Версии 3.13+ не поддерживаются базовыми ML-библиотеками (`faster-whisper`, `ctranslate2`).

## Установка и запуск (Из исходников)

**Важно:** Сборка и запуск должны производиться в изолированном виртуальном окружении.

1. Убедитесь, что у вас установлен Python 3.11 и он добавлен в PATH.
2. Создайте изолированное виртуальное окружение:
   ```powershell
   py -3.11 -m venv .venv
   ```
3. Активируйте окружение:
   ```powershell
   .\.venv\Scripts\Activate.ps1
   ```
4. Установите зависимости:
   ```powershell
   python -m pip install -r requirements.txt
   ```
5. Проверьте запуск приложения (убедитесь, что всё работает корректно перед сборкой):
   ```powershell
   python main.py
   ```

## Сборка в `.exe` (PyInstaller)

**Внимание:** Собирайте исполняемый файл **только после** успешного запуска из исходников (шаг 5 выше), находясь в активированном `.venv` со всеми установленными зависимостями.

Для упаковки приложения в автономный `.exe` файл (вместе с зависимостями `faster-whisper`), выполните команду:
```powershell
python -m PyInstaller --clean build.spec
```
Готовый исполняемый файл будет лежать в директории `dist/`.

## Конфигурация (`config.json`)
Все параметры приложения гибко настраиваются через файл `config.json`, который автоматически создается рядом с исполняемым файлом (если отсутствует).

| Параметр | Описание | По умолчанию |
| :--- | :--- | :--- |
| `hotkey` | Горячая клавиша для записи (Push-To-Talk). Поддерживает сочетания. | `"f8"` |
| `suppress_hotkey` | Подавлять ли нажатие в ОС. Если `true`, другие приложения не увидят `F8` во время диктовки. | `true` |
| `append_trailing_space` | Автоматически добавлять пробел после напечатанного текста (для слитной диктовки). | `true` |
| `language` | Язык распознавания. | `"ru"` |
| `model` | Название или путь к модели CTranslate2 (`faster-whisper`). | `"large-v3-turbo"` |
| `device` | Устройство для вычислений (`"cuda"` или `"cpu"`). | `"cuda"` |
| `compute_type` | Точность вычислений (рекомендуется `"float16"` для GPU и `"int8"` для CPU). | `"float16"` |
| `sample_rate` | Частота дискретизации микрофона в герцах. | `16000` |
| `model_dir` | Директория для кеширования скачанных моделей. | `"models"` |
| `channels` | Количество аудиоканалов (Whisper требует 1). | `1` |
| `dtype` | Внутренний формат данных для `numpy`. | `"float32"` |
| `blocksize` | Размер аудиоблока (в фреймах), получаемого с микрофона за один callback. | `2048` |
| `silence_threshold` | Порог тишины (RMS). Если общая громкость записи ниже, аудио отбрасывается. | `0.01` |
| `min_recording_duration`| Минимальная длина записи (в сек). Игнорирует случайные миллисекундные нажатия. | `0.5` |
| `max_recording_duration`| Максимально допустимая длина одной записи (защита памяти). | `60.0` |

## Известные ограничения
- При попытке диктовать в окна с повышенными привилегиями (UAC, Диспетчер задач, установщики) эмуляция клавиатуры будет заблокирована Windows. Текст не вставится, но ошибка корректно залогируется.
- Из-за глобального перехвата клавиатуры некоторые корпоративные антивирусы могут ругаться на эвристику (False Positive).
